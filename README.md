# Sales Data Analysis using Linear Regression and Time Series Analysis

## Project Overview

This project focuses on analyzing retail sales data using Python and Jupyter Notebook to uncover meaningful business insights through data visualization, Linear Regression, and Time Series Analysis techniques.

The analysis was performed on a sales dataset containing information related to product categories, sales, profit, order dates, and customer purchasing patterns. The project aims to identify sales trends, profitability patterns, and performance variations across different product sub-categories.

The project combines statistical analysis and visual storytelling to better understand business performance and decision-making opportunities.

---

# Objectives

The main objectives of this project are:

- Perform data cleaning and preprocessing
- Explore and visualize sales data
- Analyze profit and sales relationships
- Compare yearly sales performance
- Identify high-performing and low-performing sub-categories
- Study sales trends over time using Time Series Analysis
- Build a Linear Regression model for predictive analysis

---

The dataset includes the following features:

- Order Date

- Sales

- Quantity
- Region

The dataset was analyzed using Python libraries for data manipulation, visualization, and machine learning.

# 2. Data Visualization

Multiple visualizations were created to better understand sales and profit trends.

### Visualizations Included

- Sales Difference Between 2020 and 2021
- Total Sales and Profit by Sub-Category

- Rolling Average Trend

These visualizations helped identify:

- Sales growth patterns

---

# 3. Linear Regression Analysis

Linear Regression was used to analyze the relationship between Sales and Profit.

### Model Details

- Independent Variable: Sales
- Dependent Variable: Profit

The model generated:

- Regression Line
- Predicted Profit Values
- R² Score
- Regression Equation

### Key Findings

- Sales and profit show a positive relationship.
- Higher sales generally lead to increased profit.
- Certain transactions generated low profitability despite high sales values.

The regression analysis helps understand business profitability trends and predictive behavior.

---

# 4. Time Series Analysis

Time Series Analysis was performed using Order Date and Sales columns to study sales trends over time.

### Analysis Performed

- Monthly Sales Trend
- Rolling Average Analysis
- Yearly Trend Observation

### Key Findings

- Sales fluctuate over different periods.
- Rolling averages smooth short-term fluctuations.
- Long-term sales trends indicate gradual business growth.

Time Series Analysis helps identify patterns, trends, and future business opportunities.
